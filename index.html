Git and GitHub

Git Commands

  • git init => initialize a git repo
  • git add => add all the files present in current folder to the repo
  • git commit -m "" => commit process
  • git remote add origin 'url' => adding remotely to the repo
  • git branch => checking the current branch
  • git branch -M main => changing branch to 'main'
  • git push -u origin main => pushing into the main
  • git checkout -b 'name' => creating a new branch
  • git checkout 'name' => switching to another branch
  • git merge 'name' => merging one branch into another
